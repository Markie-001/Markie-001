Chapter 1

CSS Flexbox

One axis at a time — the easiest way to align and distribute items in a row or column.

🚂

Think of it like a train

Flexbox is like train carriages on a single track. They line up in a row (or column). You control how much space is between them, whether they're bunched together or spread out, and how they align vertically.

1 Turn on Flexbox

Add display: flex to the parent container. The children automatically become flex items.

.container {
  display: flex;  /* that's it! */
}

/* The children become flex items automatically */
.container div {
  width: 60px;
  height: 60px;
}

2 flex-direction — Which way do items go?

row (default) = left to right. column = top to bottom. Think of it as choosing horizontal or vertical train tracks.

3 justify-content — Space along the main axis

Controls how items are spread out along the direction they flow. This is your most-used flex property.

.container {
  display: flex;
  justify-content: space-between;  /* items spread edge to edge */
  /* other values: flex-start | flex-end | center | space-around | space-evenly */
}
4 align-items — Space on the cross axis

If justify-content is horizontal, align-items is vertical. Use center + center to perfectly center anything.

Chapter 2

CSS Grid

Two-dimensional layout — control rows AND columns simultaneously. The most powerful layout system in CSS.

🗺️

Think of it like a city map

Grid is like city blocks — you define streets going across (columns) and streets going down (rows). Then you place buildings (elements) that can span multiple blocks. Flexbox is one street; Grid is the whole city.

1 Define your grid columns & rows

Use grid-template-columns to set how many columns and how wide they are. The fr unit means "fraction of available space".

.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;  /* 3 equal columns */
  grid-template-columns: repeat(3, 1fr); /* same thing, shorter */
  grid-template-columns: 200px 1fr 1fr;  /* fixed + flexible */
  gap: 16px;
}

2 Spanning columns & rows

Items can span multiple columns or rows using grid-column and grid-row. This is how you build complex layouts.

.big-item {
  grid-column: span 2;    /* take up 2 columns */
  grid-row: span 2;       /* take up 2 rows */
}

.full-width {
  grid-column: 1 / -1;   /* span ALL columns */
}

3 grid-template-areas — Name your layout

The most readable way to define a complex layout. Name each area, then assign elements to those areas by name.

.container {
  display: grid;
  grid-template-areas:
    "header header header"
    "sidebar main main"
    "footer footer footer";
  grid-template-columns: 200px 1fr 1fr;
}

.header  { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main    { grid-area: main; }
.footer  { grid-area: footer; }
